Mes Projets

Durant mes années d'études en informatique, j'ai eu l'occasion de réaliser un certain nombre de projets dans différents domaines. J'ai également eu l'opportunité de réaliser des projets personnels. Vous pouvez donc retrouver ci-dessous le détail des projets auxquels j'ai participé.

Technologies utilisées : Flask, Bootstrap, HTML, CSS, Javascript, SQLAlchemy

2019/2020 - Projet Poly'Scape

Il s'agit d'un projet d'application web dans lequel nous étions un groupe de 8 personnes, nous avons réaliser un Escape game en ligne pour l'école Polytech. Le but était de permettre au professeurs de fournir à leurs étudiants un outil supplémentaire afin d'accompagner les étudiants dans leur apprentissage tout en ayant un aspect amusant. Lors de ce projet je m'occupe majoritairement de la partie Front-End.

2019 - Création d'un site vitrine

La visibilité sur internet étant très importante pour obtenir de nouveaux clients, je me suis occupé de créer un nouveau site vitrine pour l'entreprise APC Gaz. La société possédait déjà une maquette du site qu'ils voulaient obtenir, je me suis donc chargé de transformer la maquette en réel site web.

Technologies utilisées : HTML, CSS, PHP, JavaSript, Bootstrap
Technologies utilisées : Java, JavaFX, JDBC, MySql

2019 - Projet VAE

Il s'agit d'un projet d'application de vente aux enchères dans lequel nous étions un groupe de 6 personnes, nous avons créer une interface graphique de vente aux enchères. Notre application comprenait un module utilisateur ainsi qu'un module administrateur. Lors de ce projet je me suis occupé de la partie design de l'interface et de la liaison entre l'interface graphique et les contrôleurs.

2018 - Programmation du cluedo

Nous avions pour mission de programmer un jeu cluedo en Python par groupes de 5 étudiants. Nous avions le plateau qui s'affichait dans le terminal et les joueurs pouvait utiliser certaines commandes afin de réaliser toutes les manipulations faisables dans le jeu de société.

Technologie utilisée : Python3
